# 🎮 Word Quest (워드 퀘스트)
> **게이미피케이션 기반 영단어 학습 RPG 시스템**

영단어 학습에 RPG의 '성장'과 '게임' 요소를 결합하여, 사용자가 몰입감 있게 단어를 외우고 복습할 수 있는 안드로이드 애플리케이션입니다.

---

## 🎯 프로젝트 목표
- **학습의 재미**: 스테이지 클리어 방식의 학습을 통한 성취감 고취.
- **성장 시스템**: 학습 결과로 얻은 장비를 합성하고 강화하여 전투력 상승.
- **체계적 복습**: 오답 노트를 기반으로 한 복습 시스템으로 학습 효율 극대화.

---

## 📱 화면 및 기능 구조
### 1. Start (시작/설정)
- 앱의 대문이며 시작하기 및 환경 설정(사운드, 진동, FPS 등) 기능 제공.
### 2. Learn (학습 + 스테이지)
- 난이도별/테마별 스테이지 기반 영단어 퀴즈 진행.
### 3. Review (오답 노트)
- 학습 중 틀린 단어를 자동으로 수집하고 재학습 기회 제공.
### 4. Game (미니게임)
- 학습한 단어를 활용하여 런게임 방식의 퀴즈 미니게임을 진행.
